# API Documentation

## At a Glance

This document provides a comprehensive overview of the Gigsy MVP API, including REST/GraphQL endpoint definitions, parameters, response schemas, and example calls.

## Endpoints

### Authentication

- **POST /api/auth/login**
- **POST /api/auth/register**

### Gigs

- **GET /api/gigs**
- **GET /api/gigs/{id}**
- **POST /api/gigs**
- **PUT /api/gigs/{id}**
- **DELETE /api/gigs/{id}**
